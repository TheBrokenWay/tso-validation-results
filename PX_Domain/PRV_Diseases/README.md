# PRV Disease Manifest

**Single source of truth for the REP (Repurposing) disease universe.**

- **manifest.json** — Governed list of PRV-eligible diseases:
  - **tropical** — Nipah, Ebola, Marburg, Lassa, Dengue, Chikungunya, Zika, Rabies, TB, Malaria, Leishmaniasis, Chagas, Onchocerciasis, Schistosomiasis, Lymphatic filariasis, Dracunculiasis, Trachoma, Buruli ulcer, Yaws, Leprosy
  - **rare_pediatric** — Spinal muscular atrophy, Batten, Krabbe, Pompe, Duchenne muscular dystrophy, Congenital adrenal hyperplasia
  - **mcm** (Medical Countermeasures) — Anthrax, Smallpox, Botulism, Plague, Radiation injury

REP pipelines and prioritization run against this list.
